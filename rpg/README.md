# Meu RPG

Bem-vindo ao RPG! Aqui você encontrará instruções básicas para instalar e jogar.

## Como jogar

1. Execute `python main.py` dentro da pasta `meu_rpg`
2. Use o menu para iniciar o jogo, carregar partida ou sair
3. Durante o jogo, você controla o personagem principal, enfrenta inimigos e avança na história

## Estrutura do projeto

- `core/`: mecânicas centrais do motor do jogo
- `entities/`: personagens e inimigos
- `items/`: equipamentos e consumíveis
- `utils/`: utilitários e textos do jogo
